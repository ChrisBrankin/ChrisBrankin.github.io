Capture method

Images captured using a Toucam and X2 Barlow on the Dobsonian mounted 200mm reflector.

Focusing, exposure adjustment and actual capture all done in repeated steps each time using the few seconds available before the image drifts off the laptop screen!

Hence only a few frames available for stacking.
